Use Case 1:
Find Highest Confirmed Cases using INNER JOIN
Objective:

Specific date par jis country ke confirmed cases sabse jyada hain.

Query:
SELECT 
    c.country_name,
    cc.confirmed_cases,
    cc.date
FROM covid_cases cc
INNER JOIN countries c
ON cc.country_id = c.country_id
WHERE cc.date = '2024-01-01'
ORDER BY cc.confirmed_cases DESC
LIMIT 1;
Concepts:
INNER JOIN
WHERE
ORDER BY
LIMIT
Use Case 2:
Join COVID Deaths and Vaccination Data
Requirement:

Vaccination data missing ho tab bhi country show honi chahiye.

Isliye LEFT JOIN use hoga.

SELECT
    d.country_id,
    d.death_count,
    v.vaccination_status
FROM covid_deaths d
LEFT JOIN covid_vaccines v
ON d.country_id = v.country_id;

Output example:

country_id	death_count	vaccination_status
1	50000	Completed
2	30000	NULL
Use Case 3:
Analyze Deaths by Continent
Query:
SELECT
    ct.continent_name,
    SUM(cc.deaths) AS total_deaths
FROM covid_cases cc

INNER JOIN countries c
ON cc.country_id = c.country_id

INNER JOIN continents ct
ON c.continent_id = ct.continent_id

GROUP BY ct.continent_name;

Example Output:

continent	total_deaths
Asia	500000
Europe	700000
Use Case 4:
Calculate Average New Deaths Per Day
Query:
SELECT
    date,
    AVG(new_deaths) AS average_daily_deaths
FROM covid_cases

GROUP BY date

ORDER BY date;


Use Case 5:
Find Countries with Highest Infection Rates

Formula:

infection_rate =
(confirmed_cases / population) * 100
Query:
SELECT
    c.country_name,
    cc.confirmed_cases,
    c.population,

    (cc.confirmed_cases / c.population) * 100 
    AS infection_rate

FROM covid_cases cc

INNER JOIN countries c

ON cc.country_id = c.country_id

ORDER BY infection_rate DESC;
